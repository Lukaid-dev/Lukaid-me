## 1.3. 목록

### 순서있는 목록(번호, ol)

순서있는 목록은 숫자와 점을 사용합니다.

```
1. 첫번째
2. 두번째
3. 세번째
```

ex)

1. 첫번째
2. 두번째
3. 세번째

### 순서없는 목록(글머리 기호: `*`, `+`, `-` 지원, ul)

-> 현재 이게 지원이 잘 안되네요... 수정 예정

```
* 빨강
  * 녹색
    * 파랑

+ 빨강
  + 녹색
    + 파랑

- 빨강
  - 녹색
    - 파랑
```

ex)

- 빨강
  - 녹색
    - 파랑

* 빨강
  - 녹색
    - 파랑

- 빨강

## 1.4. 코드

### 1.4.1. 들여쓰기

```

    This is a code block.

end code block.
```

ex)

This is a normal paragraph:

    This is a code block.

end code block.


- 한줄 띄어쓰지 않으면 인식이 제대로 안되는 문제가 발생합니다.

```
This is a normal paragraph:
    This is a code block.
end code block.
```

ex)

This is a normal paragraph:
    This is a code block.
end code block.

### 1.4.1. 코드블럭

상당히 공을 들인 코드 블럭입니다. 코드 블럭은 백틱(/```)을 3번 이상 입력하고 코드 종류도 적어줍니다.

- 언어 이름을 적지 않은 경우

ex)

```
export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode,
}>) {
  return (
    <html lang="en">
      <body className={inter.className}>{children}</body>
    </html>
  );
}
```

- 언어 이름을 적은 경우

ex)

```javascript
export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode,
}>) {
  return (
    <html lang="en">
      <body className={inter.className}>{children}</body>
    </html>
  );
}

## 1.7. 강조

```
*single asterisks*
_single underscores_
**double asterisks**
__double underscores__
~~cancel line~~
```

ex)

- _single asterisks_
- _single underscores_
- **double asterisks**
- **double underscores**
- ~~cancel line~~

> `문장 중간에 사용할 경우에는 **띄어쓰기** 를 사용하는 것이 좋다.`  
> 문장 중간에 사용할 경우에는 띄어쓰기를 사용하는 것이 좋다.

## 1.9. 줄바꿈

3칸 이상 띄어쓰기(` `)를 하면 줄이 바뀐다.

```
* 줄 바꿈을 하기 위해서는 문장 마지막에서 3칸이상을 띄어쓰기해야 한다.
이렇게

* 줄 바꿈을 하기 위해서는 문장 마지막에서 3칸이상을 띄어쓰기해야 한다.___\\ 띄어쓰기
이렇게
```

ex)
